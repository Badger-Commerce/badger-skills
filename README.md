# Badger Commerce Agent Skills

Claude Code plugin providing agent skills for [Badger Commerce](https://www.bdgr.co.uk) — purpose-built extensions, component building, and site management via MCP tools.

## Installation

```bash
claude plugin install badger-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `hero` | Add and configure Hero section extensions with 5 preset layouts (Centered, Split Image, Video Background, Gradient Overlay, Minimal) |
| `json-components` | Build and edit freeform JSON components — full component catalog, semantic style tokens, data binding, and data sources |

## Prerequisites

These skills are designed to work with the Badger Commerce MCP server, which provides tools for managing extensions, media, products, collections, and site configuration.
